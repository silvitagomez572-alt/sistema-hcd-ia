4. Fórmulas Estadísticas Utilizadas
Internados únicos del mes
n = | { p : Estado="Ocupada", Área=SM, Cama ∉ Trans. } |
Se filtran filas con Estado = Ocupada y Área = Salud Mental. Se excluyen camas transitorias 53 A y 53 B. Se deduplica por codigoHC (último registro por fecha de ingreso) y luego por Documento / DNI (registro con mayor estada). El resultado n es la cantidad de filas del DataFrame consolidado.
Mayo 2026: n = 25 pacientes únicos
Porcentaje de ocupación SM
%Ocu = ( media(C_ocu_día) / C_total ) × 100
Para cada día con archivo se cuenta la cantidad de camas de CAMAS_FIJAS_SM con Estado = Ocupada. El porcentaje mensual es el promedio de esos valores diarios dividido por C_total = 18 (constante institucional).
Mayo 2026: media = 15,1 camas → 83,8 %
Promedio de estada
Ē = (1/n) · Σ Eᵢ
E_i es la estada en días del paciente i según el campo Estada de VADIGU (días desde ingreso hasta fecha del archivo). Para pacientes en múltiples archivos se conserva el valor más alto. Los valores no numéricos se descartan con pd.to_numeric(errors='coerce').
Mayo 2026: Ē = 35,6 días. Nota: refleja estada total desde el ingreso, no solo días del mes.
Giro de camas
G = n / C_pico
n = pacientes únicos del mes (fórmula 1). C_pico = valor máximo de camas fijas SM ocupadas registrado en cualquier día del mes (pico de CAMAS_FIJAS_SM con Estado = Ocupada). Refleja cuántos pacientes distintos ocuparon cada cama en promedio.
Mayo 2026: n=25, C_pico=19 → G = 1,32. Definición operativa; el giro epidemiológico clásico usa egresos/camas.

Constantes del módulo: CAMAS_FIJAS_SM = 18 camas (51A, 51B, 52A, 52B, 63A–B, 66A–B, 68A–B, 69A–B, 70A–B, 71A–B, 72A–B) · CAMAS_TRANSITORIAS = [53 A, 53 B] · TOTAL_CAMAS_FIJAS = 18.
